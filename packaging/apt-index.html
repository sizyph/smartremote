SmartRemote — APT repository

A dumb, deterministic single-GPU job dispatcher with human-in-the-loop over WhatsApp/email.

Install

echo 'deb [trusted=yes] https://sizyph.github.io/smartremote stable main' \
  | sudo tee /etc/apt/sources.list.d/smartremote.list
sudo apt-get update
sudo apt-get install smartremote

This repository is currently unsigned (hence [trusted=yes]). To verify signatures, the maintainer can add a GPG key and the line drops the [trusted=yes].

Source & documentation: github.com/sizyph/smartremote
